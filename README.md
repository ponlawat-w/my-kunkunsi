# My工工四 (くんくんしー)

簡単な工工四を書く為のツールです。

Javascriptで簡単なjQueryにて開発しております。

## 工工四とは

三線の楽譜です。

## 使い方

一般のキーボード（QWERTY）では

### 【男弦】
```
Q　→　合
W　→　乙
E　→　老
R　→　下老
```

### 【中弦】
```
A　→　四
S　→　上
D　→　中
F　→　尺
G　→　下尺
```

### 【女弦】
```
Z　→　工
X　→　五
C　→　六
V　→　七
B　→　八
N　→　九
```

### その他のホットキー
```
空白　→　〇
後退　→　一文字を削除
```
